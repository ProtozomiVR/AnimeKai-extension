# How to install
1. Paste this into the add repo box:
```html
https://raw.githubusercontent.com/ProtozomiVR/AnimeKai-extension/refs/heads/repo/index.min.json
```
2. Install the extension.



## Extra features:
1. Fixed downloads

2. Ability to enable and dissable both sub and dub servers separately. (Useful for disabling the VidPlay-1 dub server as it plays a subbed episode)

3. Added a separate 2nd version to make downloading episodes better. (Only dub and hard sub servers)

4. Dubs come first in the server list if the episode has dubs (can be disabled)
